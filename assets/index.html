    Data Engineering - Project Examples

    Pharma — Master Data Management (Oracle + Snowflake + MuleSoft)
    • What was built: MDM system maintenance and enhancement; ETL automation; integration with inbound/outbound systems via REST APIs and MuleSoft; senior lead developer role in SAFe Agile
    • Tech: Oracle DB · PL/SQL · Snowflake · MuleSoft · REST APIs · ServiceNow · BitBucket
    • Outcome: Scalable MDM with fully automated ETL; integrated with enterprise systems at scale
    Big Pharma — Snowflake Data Vault + Tableau
    • What was built: Migration of Alteryx pipelines to Snowflake data vault; ETL loading large datasets from Oracle, SQL Server, SAP HANA; Tableau dashboards; data governance with Collibra and Imuta
    • Tech: Oracle · GitLab · DBT · Snowflake · SQL Server · SAP HANA · Tableau · Collibra · Python
    • Outcome: Scalable data vault architecture; standardized ETL; governed data for analytics consumption
    Aerospace & Defense — Azure Cloud DWH + Power BI
    • What was built: Full Azure-based DWH using Data Vault; ETL via Azure Data Factory; Power BI on Azure SQL DWH; CI/CD pipeline for cube rights management
    • Tech: Azure Data Lake · ADF · Azure SQL · Azure Analysis Services · SSIS · Power BI · DevOps
    • Outcome: End-to-end cloud DWH; automated rights management via CI/CD; Agile delivery
    Telecom — Data Vault 2.0 with dbt + Snowflake
    • What was built: Data Vault 2.0 modeling (hubs, links, satellites, snapshots); dbt for transformation; GitLab CI pipelines; full version control and deployment workflow
    • Tech: SQL · Python · dbt · Snowflake · GitLab
    • Outcome: Standardized modern data engineering stack; reproducible, tested, documented transformations
    Construction — Microsoft Fabric Medallion Architecture
    • What was built: ETL/ELT flows using notebooks and pipelines following the Medallion architecture (bronze/silver/gold layers); PySpark transformations
    • Tech: Microsoft Fabric · PySpark · Spark SQL
    • Outcome: Cloud-native Lakehouse pattern on Fabric; scalable layered data processing
    Banking — DWH Outsourcing Architecture to 24 Countries
    • What was built: As-is / to-be design for outsourcing bank DWH across 24 countries; managed onshore + offshore teams; pilot delivery in Risk and Marketing domains
    • Tech: Teradata · SQL · HPE ALM
    • Outcome: Multi-country DWH rollout architecture; pilot delivered in Risk and Marketing
    Tagetik EPM — Finance Planning & Consolidation
    • What was built: Multiple Tagetik implementations across EPM modules: complex calculation logics, allocations, virtual datasets, DTPs, ETLs, workflows; historical data migration; security model updates
    • Tech: CCH Tagetik · SAP HANA · PostgreSQL · Databricks
    • Outcome: Finance planning, consolidation, and reporting platforms across Food, Energy, Retail
    GenAI Evaluation Platform (Databricks + Azure OpenAI)
    • What was built: GenAI evaluation and benchmarking system: test cases to evaluate AI response quality, consistency, accuracy; prompt engineering; result analysis; experiment tracking
    • Tech: Databricks · SQL · Python · Azure OpenAI
    • Outcome: Reproducible GenAI evaluation framework enabling quality validation across AI models

    Data Analytics - Project Examples

    Automotive — Large-Scale KPI Reporting Ecosystem
    • What was built: 60+ reports and dashboards across 5 operational streams; 30+ data sources integrated; full semantic model with KPI/SLA framework; multi-year ownership
    • Tech: Power BI · SharePoint · OneDrive · Teams B2B
    • Outcome: Daily-refresh reliability maintained 2+ years; full KPI/SLA framework owned by analytics team
    Consumer Goods — Automated Reporting via REST API Pipeline
    • What was built: Python ETL extracting from Jira and SonarQube REST APIs; data staged on SharePoint; Power BI on top; fully automated scheduled pipeline
    • Tech: Python · Power BI · SharePoint · REST APIs · Jira · SonarQube · Excel
    • Outcome: Automated productivity KPI reporting; zero manual data collection
    Renewable Energy — Legacy Migration from Excel + PPT to Power BI
    • What was built: Replaced shared Excel files with SharePoint lists for annual planning; replaced email-based PowerPoint reporting with Power BI + SharePoint; introduced Power Apps forms for data entry
    • Tech: Power BI · SharePoint · Power Apps
    • Outcome: Improved data quality, reduced manual effort, audit trail on all inputs
    FMCG — Retailer KPI, Sales & Capacity Dashboards
    • What was built: Business reports for market analysis; Power BI Embedded datasets for retailer data; SMS campaign performance reports; discrepancy detection tools; automated capacity monitoring
    • Tech: Power BI · Power BI Embedded · Log Analytics · DevOps · Synapse Serverless · ServiceNow · Excel
    • Outcome: KPI tracking for operations managers; automated alert-based capacity monitoring
    Travel — Azure Cloud Stack Reporting (Synapse + CosmosDB)
    • What was built: Power BI layer connected to CosmosDB via SQL queries; Synapse views created and linked to Power BI; star schema design; data validation between DB layer and report outputs
    • Tech: Power BI · Azure Synapse · CosmosDB · Azure DevOps · SharePoint
    • Outcome: Scalable cloud-native reporting layer integrated with modern Azure data stack
    Finance & Accounting — Excel to Power BI Migration
    • What was built: Full migration of tables and charts from Excel to Power BI; data cleaning and accuracy checks; new measures, conditional columns; monthly maintenance process
    • Tech: Power BI · Excel · SharePoint
    • Outcome: Standardized reporting; reduced Excel dependency; maintainable update process
    Telecom — HR Data Staging and Validation in Power BI
    • What was built: Power BI used as final staging layer for HR data migration; scalable process to transform, validate and correct historical data from source systems
    • Tech: Power BI · SQL
    • Outcome: Controlled data migration with complete validation at report layer
