## Certifications

### Visual Insights Domain

| Certification | Provider | Level | Relevance |
|---|---|---|---|
| PL-300: Microsoft Power BI Data Analyst | Microsoft | Intermediate | Core certification for the Visual Insights domain. Validates Power BI report authoring, data modeling, and analytics. Recommended at L10 (Senior Analyst). |
| PL-900: Microsoft Power Platform Fundamentals | Microsoft | Foundational | Covers Power BI, Power Apps, Power Automate. Good entry-level cert for Associates (L12). |
| DP-900: Microsoft Azure Data Fundamentals | Microsoft | Foundational | Cloud data concepts, relational and non-relational data, analytics workloads. Recommended early in career for both domains. |
| AZ-900: Microsoft Azure Fundamentals | Microsoft | Foundational | General Azure literacy. Useful baseline before pursuing data-specific Azure certs. |

### Data Engineering Domain

| Certification | Provider | Level | Relevance |
|---|---|---|---|
| DP-203: Azure Data Engineer Associate | Microsoft | Intermediate | Core certification for the Data Engineering domain. Covers Azure Synapse, Data Factory, Databricks, storage. Recommended at L10. |
| DP-100: Azure Data Scientist Associate | Microsoft | Intermediate | Azure Machine Learning, MLOps, model deployment. Relevant for DE practitioners moving toward ML workloads. |
| AZ-900: Microsoft Azure Fundamentals | Microsoft | Foundational | Recommended first Azure cert for all new joiners. |
| DP-900: Microsoft Azure Data Fundamentals | Microsoft | Foundational | Quick path to foundational data concepts on Azure. |
| AWS Cloud Practitioner (CLF-C02) | AWS | Foundational | AWS literacy baseline. Relevant on AWS-platform projects. |
| AWS Certified Data Engineer — Associate (DEA-C01) | AWS | Intermediate | Validates AWS data engineering skills: Glue, Redshift, Kinesis, S3 patterns. |
| GCP Professional Data Engineer | Google Cloud | Professional | Covers BigQuery, Dataflow, Pub/Sub, Dataproc. Relevant on GCP-platform projects. |
| Databricks Certified Associate Developer for Apache Spark | Databricks | Associate | Validates PySpark programming fundamentals. Good first Databricks cert. |
| Databricks Certified Data Engineer Associate | Databricks | Associate | Covers Delta Lake, Databricks workflows, Unity Catalog basics. Recommended for DE practitioners on Databricks projects. |
| Databricks Certified Data Engineer Professional | Databricks | Professional | Advanced Databricks patterns, performance tuning, production pipelines. |
| SnowPro Core | Snowflake | Core | Validates working knowledge of Snowflake platform. Recommended for any project using Snowflake. |
| SnowPro Advanced: Architect | Snowflake | Advanced | Snowflake architecture, design patterns, and optimization. |
