# Senior Analyst — Data Engineering

> **Career Level:** L10 | **Domain:** Data Engineering

## Overview

A Senior Analyst Data Engineer owns end-to-end data platform workstreams — from architecture input through pipeline implementation, testing, and deployment. They take accountability for the technical quality of their deliverables, actively contribute to data modelling decisions, and begin mentoring junior colleagues. At L10, the engineer deepens expertise in big data tooling (Apache Spark, Databricks) and advances cloud platform knowledge from foundational to practitioner level.

## Learning Goals

- Design and own data flows across the full pipeline lifecycle (ingestion, transformation, serving)
- Apply data modelling techniques (dimensional modelling, Data Vault basics)
- Use Apache Spark on Databricks for distributed data processing
- Configure and administer Databricks workspaces and integrations
- Apply business analytics and intelligence principles to shape data products
- Work with Scala for Spark-based processing (foundational level)
- Implement DevOps practices — CI/CD, testing automation, deployment pipelines
- Mentor L11/L12 engineers through code review and pairing
- Obtain an Azure or GCP professional-level data certification

## Key Skills

**Technical Skills** (from Data Engineering and Data Architecture specializations)
- Apache Spark (PySpark, DataFrame API)
- Databricks (administration, integrations, ETL pipelines)
- Data Modelling Techniques and Methodologies
- Azure Synapse Analytics
- Scala Language Fundamentals
- Business Analytics and Intelligence
- CI/CD and DevOps Foundations
- Big Data Architecture
- Data Quality Management

**Soft Skills**
- Technical leadership within a workstream
- Translating data requirements into technical designs
- Code review and constructive feedback
- Communicating data platform capabilities to non-technical stakeholders

## Tools & Technologies

- **Big data processing:** Apache Spark (PySpark), Scala
- **Lakehouse platform:** Databricks (clusters, notebooks, jobs, Delta Lake basics)
- **Cloud data services:** Azure Synapse Analytics, Azure Data Factory, AWS Glue, Amazon Redshift, Google BigQuery
- **Orchestration:** introduction to Apache Airflow concepts
- **Data modelling:** dimensional modelling, Data Vault awareness
- **DevOps & CI/CD:** Azure DevOps, Git branching strategies, automated testing
- **Scripting:** Python (advanced — pySpark, pyiceberg, pandas)
- **Query & analytics:** SQL (expert level)
