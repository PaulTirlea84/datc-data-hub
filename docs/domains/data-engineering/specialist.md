# Specialist — Data Engineering

> **Career Level:** L9 | **Domain:** Data Engineering

## Overview

A Specialist Data Engineer holds deep technical expertise in one or more modern data platforms — most commonly Databricks, Snowflake, or advanced cloud architectures (AWS, Azure, GCP). They set technical standards for the team, own complex solution designs within their specialisation, drive platform adoption, and serve as the go-to technical authority for their area. Specialists may also take on team lead responsibilities, leading delivery of a workstream or squad while continuing hands-on technical work.

## Learning Goals

- Architect and implement end-to-end data platforms on Databricks or Snowflake
- Apply advanced cloud architecture patterns (Lakehouse, Data Mesh, Data Fabric)
- Lead Python data engineering initiatives — pySpark, pyiceberg, Pandas at scale
- Design and implement data governance frameworks using Unity Catalog (Databricks) or Snowflake role-based access
- Configure infrastructure as code (Terraform) for data platform provisioning
- Define team standards for pipeline design, testing, and documentation
- Mentor and technically lead a squad of L11/L10 engineers
- Drive or contribute to pre-sales and proposal activities
- Obtain AWS Solutions Architect Professional or equivalent advanced certification

## Key Skills

**Technical Skills** (from Databricks, Snowflake, and Data Engineering specializations)
- Databricks Unified Data Analytics Platform (administration, Unity Catalog, Lakeflow, Asset Bundles)
- Snowflake Data Warehouse (Snowpark, Snowpipe, warehouse design, performance tuning, governance)
- Apache Spark (advanced — streaming, optimisation, Spark SQL)
- Apache Kafka (streaming data ingestion)
- Apache Airflow (workflow orchestration)
- Terraform / Infrastructure as Code
- Docker (containerised data workloads)
- Data Lakehouse Architecture
- Data Mesh principles and implementation
- Advanced Python (pySpark, pyiceberg, pandas, numpy)
- dbt (data build tool) — transform, model, test, document

**Soft Skills**
- Technical leadership and setting engineering standards
- Stakeholder communication on platform strategy and delivery
- Running knowledge-sharing sessions and internal capability building
- Estimating and scoping data engineering deliverables

## Tools & Technologies

- **Lakehouse platform:** Databricks (Unity Catalog, Lakeflow, Delta Lake, Spark Declarative Pipelines, Asset Bundles)
- **Cloud data warehouse:** Snowflake (Snowpark, Snowpipe, Snowpipe Streaming, Polaris Catalog)
- **Streaming & messaging:** Apache Kafka, Amazon Kinesis
- **Orchestration:** Apache Airflow, Databricks Workflows
- **Transformation layer:** dbt (data build tool)
- **Infrastructure as code:** Terraform
- **Containerisation:** Docker
- **Cloud platforms (advanced):** Azure Synapse Analytics, Azure Data Factory, AWS Glue, Amazon EMR, Amazon Redshift, Google BigQuery
- **Python advanced stack:** pySpark, pyiceberg, pandas, numpy
- **Governance & catalog:** Databricks Unity Catalog, Collibra (awareness), Alation (awareness)
