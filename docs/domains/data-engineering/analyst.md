# Analyst — Data Engineering

> **Career Level:** L11 | **Domain:** Data Engineering

## Overview

An Analyst Data Engineer independently develops and maintains data pipelines, taking end-to-end ownership of defined data flows from source ingestion through transformation to target load. They work on real project deliverables — typically contributing to cloud data engineering tasks on Azure, AWS, or GCP — and produce tested, documented code with limited guidance. They begin specialising in one cloud platform and deepen their SQL and Python proficiency significantly while learning foundational data warehousing patterns.

## Learning Goals

- Build and maintain production-grade ETL/ELT pipelines on a cloud platform
- Write advanced SQL (window functions, CTEs, query optimisation, stored procedures)
- Develop Python scripts for data manipulation and pipeline automation
- Understand and implement data warehousing patterns (star schema, staging layers, SCDs)
- Work with cloud-native storage and database services (Azure Blob / S3 / GCS, Azure SQL / Redshift / BigQuery)
- Contribute to CI/CD workflows using Git and Azure DevOps
- Apply data quality checks and document pipelines for handover
- Prepare for and obtain a cloud data associate-level certification

## Key Skills

**Technical Skills** (from Data Engineering specialization)
- Advanced SQL (CTEs, window functions, query optimisation)
- Python (intermediate — Pandas, file I/O, error handling, API calls)
- Cloud database services (Azure SQL Database, Amazon RDS, Google Cloud SQL)
- Cloud storage (Azure Blob Storage, Amazon S3, Google Cloud Storage)
- ETL/ELT pipeline development
- Data Warehousing concepts (star schema, staging layers, slowly changing dimensions)
- Git branching and CI/CD basics
- Data Quality Tools and Methods

**Soft Skills**
- Independent task delivery with clear written communication
- Documenting own work (pipeline docs, data dictionaries)
- Raising blockers early and proposing solutions
- Cross-functional collaboration with analysts and architects

## Tools & Technologies

- **Query & scripting:** SQL (advanced), Python (Pandas, SQLAlchemy, requests)
- **Cloud platforms:** Microsoft Azure (Blob Storage, Azure SQL, ADF basics), AWS (S3, RDS, Glue basics), GCP (BigQuery basics, Cloud Storage)
- **Version control & CI/CD:** Git, Azure DevOps Pipelines
- **Data modelling:** ERD tools, star/snowflake schema design
- **IDE:** Visual Studio Code, Jupyter Notebooks
- **Collaboration:** Jira, Confluence, Teams

## Certifications in Scope

- Microsoft Azure Data Fundamentals (DP-900)
- AWS Certified Data Engineer — Associate
- GCP Associate Cloud Engineer
