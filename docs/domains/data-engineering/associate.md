# Associate — Data Engineering

> **Career Level:** L12 | **Domain:** Data Engineering

## Overview

An Associate Data Engineer works under close supervision to build foundational skills in SQL, Python, and cloud basics. Day-to-day work involves executing well-defined tasks such as writing simple queries, assisting with data ingestion jobs, and learning the team's toolchain. At this level the goal is absorption — understanding how data moves through a pipeline, how environments are set up, and how to contribute reliably to a team delivery.

## Learning Goals

- Write and optimise basic to intermediate SQL queries (SELECT, JOIN, GROUP BY, window functions)
- Understand core Python constructs and use Pandas and NumPy for data manipulation
- Explain the difference between ETL and ELT and describe when each is used
- Navigate the fundamentals of at least one cloud platform (Azure, AWS, or GCP)
- Use Git for version control — branches, commits, pull requests
- Understand data warehouse concepts: schemas, tables, staging vs. production layers
- Complete foundational cloud certification (AZ-900 or AWS Cloud Practitioner or GCP Cloud Digital Leader)
- Follow data quality and documentation standards defined by senior engineers

## Key Skills

**Technical Skills**
- SQL (query writing, joins, aggregations)
- Python basics (Pandas, NumPy)
- ETL concepts and basic pipeline construction
- Cloud fundamentals — Azure / AWS / GCP
- Git and version control
- Relational database concepts
- Data Governance Literacy
- Data Visualization Literacy

**Soft Skills**
- Active listening and asking clarifying questions
- Written status updates and task tracking
- Willingness to learn from code reviews
- Time management within sprint cycles
