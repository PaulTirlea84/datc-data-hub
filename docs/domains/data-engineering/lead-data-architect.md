# Lead Data Architect — Data Engineering

> **Career Level:** L8 Bridge | **Domain:** Data Engineering

## Overview

A Lead Data Architect defines the enterprise data vision for clients and the Accenture practice. They operate at the intersection of business strategy and technical architecture — advising C-suite and senior client stakeholders on data platform modernisation, cloud migration strategy, and data governance operating models. They own the end-to-end architectural direction across engagements, set platform standards for the DATC Data Engineering community, drive thought leadership, and act as the final technical authority on complex or ambiguous architectural decisions.

## Learning Goals

- Define enterprise data strategy and multi-year data platform roadmaps for clients
- Lead data governance transformation: policies, operating models, data ownership, lineage at scale
- Design and govern Data Mesh implementations across large organisations
- Apply enterprise integration architecture patterns at cloud scale
- Evaluate and recommend emerging data technologies (Data Fabric, Lakehouse evolution, streaming-first architectures, AI-ready data platforms)
- Drive Accenture DATC practice development: standards, reusable assets, capability frameworks
- Advise on data privacy, security architecture, and regulatory compliance (GDPR, DORA)
- Mentor Data Architects and Specialists, shaping the technical community
- Represent the practice in external forums, client proposals, and industry conversations

## Tools & Technologies

- **Full enterprise data stack:** Databricks (Unity Catalog, governance), Snowflake (Polaris Catalog, Cortex), Azure Synapse Analytics, Amazon Redshift, Google BigQuery, Microsoft Fabric
- **Governance & catalog:** Collibra Data Governance, Alation Data Catalog, Accenture AIP Data Governance Services (ADGS), Accenture Unified Data Model (AUDM), BigID, Amazon Macie
- **Streaming & real-time:** Apache Kafka, Amazon Kinesis, Azure Event Hubs, Apache Flink (awareness)
- **Integration patterns:** Azure Data Factory, AWS Glue, Denodo Data Virtualization, IBM InfoSphere DataStage
- **Architecture tools:** ERwin, draw.io, Lucidchart, Azure Architecture Center, AWS Well-Architected Framework
- **NoSQL & graph:** AWS Neptune, MongoDB, Knowledge Graph platforms
- **AI-ready data:** Vector databases, RAG infrastructure patterns, ML feature stores
- **Compliance & security:** BigID, Amazon Macie, Azure Purview, data encryption and masking standards
