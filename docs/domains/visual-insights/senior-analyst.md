# Senior Analyst — Visual Insights

> **Career Level:** L10 | **Domain:** Visual Insights (Power BI & Data Visualization)

## Overview

The Senior Analyst owns end-to-end BI solutions on complex, multi-source projects. You are accountable not just for delivery but for solution quality: you design the semantic model architecture, define development standards for the team, and validate that DAX logic and data accuracy hold up under production conditions. You are a trusted point of contact for client stakeholders, input into architecture decisions, and you actively mentor Analysts and Associates. At this level you begin bridging the gap between pure development and practice-level thinking — proposing improvements to how the team works, not just what it delivers.

## Learning Goals

- Architect full end-to-end Power BI solutions: from source system integration through semantic model to published app
- Master advanced DAX: virtual tables, CALCULATE modifiers (KEEPFILTERS, REMOVEFILTERS, ALL/ALLEXCEPT), many-to-many relationships, and bi-directional filtering trade-offs
- Design scalable Power Query pipelines: incremental refresh, folding optimization, and custom M functions
- Apply enterprise semantic model patterns: composite models, DirectQuery vs Import trade-offs, large dataset optimization
- Lead UAT and data validation against source systems; communicate discrepancies clearly to stakeholders
- Set and enforce team standards for naming conventions, version control (PBIP format, Azure DevOps), and documentation
- Understand Power BI embedding basics (service principal, REST API) and when to recommend it
- Coach junior team members: code review, onboarding support, and structured knowledge transfer
