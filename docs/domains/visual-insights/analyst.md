# Analyst — Visual Insights

> **Career Level:** L11 | **Domain:** Visual Insights (Power BI & Data Visualization)

## Overview

The Analyst owns report delivery end-to-end within a well-scoped project context. You gather requirements directly from the client or stakeholder, design the data model, write the DAX measures, build the report, test it, and deploy it to the Power BI Service — without needing step-by-step supervision. You are the primary developer on individual feature requests and small projects, expected to proactively flag data quality issues, performance problems, and requirement gaps before they become delivery risks. At this level you also begin participating in peer reviews and mentoring incoming Associates.

## Learning Goals

- Lead the full Power BI development lifecycle: requirements gathering, solution design, development, UAT support, and deployment
- Write intermediate DAX: time intelligence patterns (YTD, MTD, SPLY), CALCULATE with complex filters, iterators (SUMX, AVERAGEX), and context transition
- Design and optimize Power Query transformations using M code; handle parameters and dynamic queries
- Build and document a robust semantic model: relationships, calculated tables, hierarchies, and field formatting
- Implement row-level security (RLS) and validate it against stakeholder requirements
- Configure Power BI Service workspaces, apps, and deployment pipelines (Dev / Test / Prod)
- Identify and resolve report performance bottlenecks; apply best-practice model design
- Apply visual storytelling principles: layout, color hierarchy, tooltip design, and drill-through navigation
