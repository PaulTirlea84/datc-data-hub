# Associate — Visual Insights

> **Career Level:** L12 | **Domain:** Visual Insights (Power BI & Data Visualization)

## Overview

The Associate is the entry point into the Visual Insights domain. Day-to-day work involves learning the fundamentals of Power BI under close supervision — connecting to data sources, building basic reports, and understanding how the BI development lifecycle works. At this level you are a supported contributor: you shadow senior colleagues, take on well-defined tasks within an existing solution, and build the foundational habits (clean data models, consistent formatting, source-controlled files) that will carry you through the career ladder.

## Learning Goals

- Understand the core components of Power BI: Desktop, Power Query (M), Power Pivot (DAX), Power View, and the Power BI Service
- Load data from common sources (Excel, SharePoint, SQL databases, cloud connectors) and perform basic transformations in Power Query
- Build a simple star-schema data model with correct table relationships
- Write basic DAX measures (SUM, AVERAGE, CALCULATE, FILTER, simple time intelligence)
- Create a polished, client-ready report page: consistent visuals, slicers, drill-through, and row-level security basics
- Publish a report to the Power BI Service and configure a scheduled refresh
- Follow the team's development standards: naming conventions, version control, and workspace hygiene
- Understand when to use a calculated column vs. a measure
