# Visual Insights — Domain Overview

The **Visual Insights** domain covers front-end data delivery within DEVI. Practitioners in this domain develop, migrate, deploy, and maintain data-driven insight interfaces that drive adoption and decision making. They build scalable and open BI architecture to provide cross-enterprise visibility, integrate security and data privacy, and translate complex data into clear, actionable stories for business stakeholders.

## What Visual Insights Does

Visual Insights maps to three core Accenture specializations:

**BI Engineering** — develop and maintain commercial and bespoke knowledge engine interfaces. Build open BI architecture for cross-enterprise visibility. Skills: BI Architecture Design, BI Reporting Tools, Data Modeling Techniques and Methodologies, Performance & Scalability Design, Data Mapping & Transformation, Data Architecture Implementation & Test, Data Analysis & Interpretation.

**Data Visualization** — create compelling visual representations of complex data sets. Translate raw data into dashboards and insights that enhance understanding and aid decision making. Skills: Advanced Data Visualization Techniques, Data Visualization, Data Analysis & Interpretation, BI Strategy, Information Architectures, Data Value Articulation, Business Intelligence (BI) Reporting Tools.

**Enterprise BI Architecture & Engineering** — design and implement systems and frameworks that enable organizations to collect, manage, analyze, and visualize data for informed decision-making. Create robust, scalable infrastructure to support BI initiatives at enterprise scale. Skills: BI Architecture Design, Data Architecture Implementation & Test, Data Modeling Techniques and Methodologies, Performance & Scalability Design, Data Profiling & Discovery, Platform Engineering, Data Mapping & Transformation, Machine Learning Architecture Design.

**Vis & Interaction Science** — design and create data and insight interfaces that drive adoption, decision-making, and behavior change. Build organizational data literacy through prototypes and experiences. Skills: BI Strategy, Data Analysis & Interpretation, Interaction Design, Data Visualization, Information Architectures, Human-Centered Design.

## Key Tools and Technologies

- **Microsoft Power Business Intelligence (BI)** — primary BI platform
- **Microsoft Copilot for Power BI** — AI-assisted report authoring
- **Microsoft SQL Server Analysis Services (SSAS)** — enterprise semantic modeling
- **Grafana** — operational dashboards and monitoring visualization
- **D3.js** — custom interactive data visualization
- **Qlik Sense / Qlik Insight Advisor / Qlik Compose** — Qlik BI suite
- **Google Looker Data Platform** — cloud-native BI
- **IBM Cognos Analytics / IBM Cognos Reporting / IBM Cognos TM1** — IBM BI stack
- **MicroStrategy Business Intelligence** — enterprise BI reporting
- **Amazon Quicksight** — AWS-native BI
- **AtScale** — semantic layer platform
- **Palantir Foundry** — integrated analytics platform
