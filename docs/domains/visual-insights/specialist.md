# Specialist — Visual Insights

> **Career Level:** L9 | **Domain:** Visual Insights (Power BI & Data Visualization)

## Overview

The Specialist is a recognized deep expert in the Visual Insights domain. You set the technical direction for the BI practice at the team level: defining standards, reviewing architectures proposed by Senior Analysts, and driving adoption of advanced capabilities (embedding, Premium capacity optimization, custom visuals, Fabric integration). You have cross-team influence — you are pulled into solution design conversations before development starts, and you are accountable for the quality bar across multiple concurrent projects. The Specialist role is also the primary bridge between Visual Insights and the broader Data & AI domain: you understand how BI fits into a modern data platform and can have credible technical conversations with Data Engineers and Architects.

## Learning Goals

- Define and own the team's BI development standards: model design rules, DAX patterns library, naming conventions, and ALM process
- Master Power BI Premium and Fabric: capacity planning, large semantic models, Direct Lake mode, and Dataflows Gen2
- Lead complex embedding implementations: service principal, multi-tenant architectures, Power BI REST API automation
- Evaluate and adopt new Power BI capabilities (Copilot, AI visuals, composite models at scale) with an eye on production readiness
- Design BI governance: workspace taxonomy, dataset ownership, certified dataset program, sensitivity labels
- Contribute to cross-domain architecture: how Power BI connects to Azure Synapse, Fabric Lakehouse, Databricks, or Snowflake
- Coach and technically review work from Analysts and Senior Analysts; conduct structured code and model reviews
- Represent BI domain in client pre-sales or solution design workshops
